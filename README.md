## 📌 About This Repository

This repository is a growing collection of production-ready RPA (Robotic Process Automation) bots built to automate real-world business processes — from document understanding to LinkedIn outreach, admin panels, and insurance systems.

Whether you're a developer, startup, operations manager, or automation enthusiast, this repo provides plug-and-play scripts and workflows to **automate tedious tasks** and **boost productivity**.

## 🧰 Tech Stack & Capabilities
🐍 Python – Core language for scripting and automation logic

🧭 Selenium – Web browser automation and interaction

🖱️ PyAutoGUI – Mouse/keyboard GUI automation

📄 PyPDF2 – PDF reading, writing, and manipulation

🧠 Tesseract OCR – Optical Character Recognition for scanned documents

🧩 UiPath – External RPA integration (attended/unattended bots)

📝 OpenCV & PIL – Visual automation and image processing

📦 Pandas – Data transformation and structured output

➗ NumPy – Numerical computing and matrix operations

🧠 Natural Language Processing (NLP) – Smart parsing, classification, and email understanding

🤖 AI Models – ML/LLM logic for intelligent automation

🔍 Computer Vision – Detecting and reading visual screen elements

📊 Document Training Pipelines – Custom models for document understanding and key-value extraction

🧬 Hybrid Automation – Combining rule-based logic with AI intelligence

💾 MySQL / SQLite – Structured data storage and retrieval

🗄️ Database Automation – End-to-end database workflows (CRUD, syncing, etc.)

🖼️ Image Classification – Tagging and recognizing visual data

📝 Text Extraction – From PDFs, scans, screenshots, and images

📈 Power BI Integration – Dashboard generation and analytics from automation output
